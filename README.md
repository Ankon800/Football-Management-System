# ⚽ Football Management System (Java)

The **Football Management System** is a simple Java-based project that allows users to book and manage tickets for football matches. The system simulates a basic ticket purchasing process where users can view available matches, select seats, and book tickets.

This project demonstrates the use of **Java programming** and **Object-Oriented Programming (OOP)** concepts to build a simple management system for football match ticket reservations.

---

## 🚀 Features

- View available football matches  
- Book tickets for selected matches  
- Display ticket information  
- Manage booking details  
- Simple and user-friendly interface  

---

## 🛠 Technologies Used

- **Java**
- **Object-Oriented Programming (OOP)**
- Basic data handling

---

## 🎯 Purpose

The main purpose of this project is to practice **Java programming concepts** by developing a simple football management and ticket booking system. It helps beginners understand how booking systems and basic management applications are developed.
